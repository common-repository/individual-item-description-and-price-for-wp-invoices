== Description ==

It's stable add-on plugin provide option for WPinvoice to change individual item description and price, it's great add on feature who using wp invoices plugin to add description and price and it's helps to your account.

https://vimeo.com/380875997

= Features =

* The Plugin can be installed and activated without PHP notices, warnings or errors
* Item now has a new field "Description" which is a RichText editor (minimum: bold, italic, ul, li, link)
* Item can be added to invoice and quote and have
  - all the default fields of an item
  - a field "Individual Description" which is prefilled from the existing Item
* The invoice/quote the **individual description** is saved for the particular invoice only.
* The prefilled individual description is not overridden in the predefined product item.
* The prefilled individual price is not overridden in the predefined product item.
* The individual description and price is shown in whenever necessary (html template, pdf).
* [WPML](https://wpml.org/) Compatibility and translatable plugin
* Loco translatable plugin

== Installation ==

1. Upload all the files into your wp-content/plugins directory, be sure to put them into a folder called "wp-invoice-add-on"
2. Activate the plugin at the plugin administration page
3. Follow set-up steps on main Invoice page
